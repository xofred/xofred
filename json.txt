--json

object
    用大括号包住,由名字(用字符串表示):值组成(大部分情况冒号前后可以有空格)，元素之间用逗号分隔，例如
{
  "name": "My First Extension",
  "version": "1.0",
  "manifest_version": 2,
}

值可以是
    字符串,数字,object,数组,true,false,null

数组
    用中括号包住，由值组成，元素之间用逗号分隔

字符串（参考C语言）
    由零个或多个Unicode字符组成，用双引号包住，反斜杠作为转义字符

转义字符
    \"	双引号
    \\	反斜杠
    \/  斜杠
    \b	退格
    \f	格式化
    \n	换行
    \r	回到行首
    \t  水平制表符
    \u	十六进制数字，由4位表示

数字（不使用八进制和十六进制格式，其他和C语言一样）
